# 系统优化助手 v1.1.0 发布说明

发布日期：2025-12-18

## 🎯 重大更新

本版本对软件进行了全面改造，重点提升隐蔽性和易用性。

## ✨ 新增功能

### 1. 系统托盘功能
- **默认隐藏运行**：程序启动后自动隐藏到系统托盘
- **托盘菜单**：
  - 进入：显示主窗口
  - 快捷键开关：启用/禁用全局快捷键
  - 关闭：退出程序
- **静默运行**：所有操作通过托盘图标进行

### 2. 全局快捷键 (Ctrl+Alt+L)
- **系统级快捷键**：在任何应用程序中都可触发
- **快速优化**：无需打开窗口即可启动系统优化
- **可开关**：通过托盘菜单可启用/禁用快捷键

### 3. 精简UI界面
- **移除定时任务标签页**：只保留核心优化功能
- **迷惑性表述**：
  - "立即锁死系统" → "立即优化系统"
  - "解锁密码" → "恢复密码"
  - "内核级屏蔽" → "系统优化"
- **简洁布局**：更加直观易用

## 🚀 优化改进

### 1. 静默运行
- 优化完成后不再弹出提示窗口
- 系统自动恢复，完全静默
- 提升用户体验和隐蔽性

### 2. 窗口管理
- 关闭窗口改为隐藏到托盘
- 只能通过托盘菜单访问主界面
- 更符合后台工具使用习惯

### 3. 启动逻辑
- 程序启动后直接隐藏
- 首次运行引导优化
- 更新迷惑性欢迎信息

## 📝 配置变更

新增配置项：
```json
{
  "hotkey_enabled": true  // 全局快捷键开关
}
```

## ⚙️ 技术改进

### 依赖更新
需要新增以下依赖：
- `pystray`: 系统托盘支持
- `Pillow (PIL)`: 图标生成

### 安装依赖
```bash
pip install pystray pillow
```

### 全局快捷键实现
- 使用 Windows Hook 实现系统级快捷键
- Ctrl+Alt+L 组合键监听
- 支持开关控制

## 🔧 使用指南

### 首次使用
1. 运行程序（需管理员权限）
2. 程序自动隐藏到托盘（右下角）
3. 右键托盘图标查看菜单

### 优化系统
**方式一**：使用快捷键
- 按 `Ctrl+Alt+L` 立即启动优化

**方式二**：使用界面
1. 右键托盘图标 → 进入
2. 设置恢复密码
3. 点击"立即优化系统"

### 恢复系统
- 优化期间盲打设置的密码
- 系统自动恢复正常
- 不会弹出任何提示

### 快捷键开关
- 右键托盘图标
- 点击"快捷键：✓ 开启/✗ 关闭"
- 切换快捷键启用状态

## ⚠️ 注意事项

### 重要提示
1. **记住密码**：优化后只能通过密码恢复
2. **管理员权限**：必须以管理员身份运行
3. **快捷键冲突**：如与其他软件冲突，可通过托盘菜单关闭

### 升级须知
- 从 v1.0.x 升级无需卸载
- 配置文件自动兼容
- 建议重新安装依赖
